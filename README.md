# 🎬 Movie Recommender Engine

A Machine Learning-based Movie Recommendation System built with Python and Streamlit. The application recommends similar movies based on content similarity and displays movie posters, ratings, and details using the TMDB API.

## 🚀 Features

- Recommend movies similar to the selected movie
- Displays movie posters and ratings
- Fetches real-time movie details using TMDB API
- Interactive web interface built with Streamlit
- Fast recommendation using cosine similarity

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Pickle
- TMDB API

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Sukruth08/Movie-Recommender-Engine.git
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run movie_recommender_streamlit.py
```

## 📌 Note

Large dataset files and similarity matrices are not included because GitHub has file size limitations. Generate them locally before running the project.
